# ויקי פונדקאות לגייז

!!! info "אודות הויקי"
    ויקי זה מסכם ידע שנאסף בקבוצת וואטסאפ קהילתית בנושא תהליכי פונדקאות.
    כל המידע **אנונימי** ומשקף דעות ונסיון של חברי הקבוצה לאורך השנים.
    המידע אינו תחליף לייעוץ משפטי, רפואי או פיננסי מקצועי.

## איך להשתמש

הויקי מחולק לנושאים בתפריט הצד: לפי מדינה, סוכנויות וספקים, היבטים משפטיים,
גיור, כסף ותשלומים, שלבי התהליך ועוד. כל עמוד מציג את ההסכמה הרווחת לצד דעות
מנוגדות, עם ציון כמה משתתפים תמכו בכל עמדה ומתי (חודש ושנה) נאמרו הדברים.

!!! warning "בבנייה"
    התוכן נבנה בהדרגה. עמודים מאושרים מתווספים לתפריט לאחר בדיקה אנושית.
